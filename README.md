# HyperOS 3 Lite Blur Control Center

A lightweight systemless module designed to enable the **Lite Blur Control Center** on **HyperOS 3 (Android 15)** devices. Fully compatible with both **Magisk** and **KernelSU**, this module enhances the Control Center by introducing a premium blur-style interface and smooth opening and closing animations while maintaining excellent performance and stability.

Unlike flagship devices that utilize resource-intensive real-time blur, this module implements a **Lite Blur** effect that delivers a clean and modern visual experience without the additional GPU overhead. It is optimized for users who want a more refined Control Center appearance while preserving battery life and system responsiveness.

---

## Features

- ✨ Enables Lite Blur Control Center
- 🎬 Smooth Control Center opening and closing animations
- ⚡ Lightweight and optimized for daily use
- 🔒 Compatible with both **Magisk** and **KernelSU**
- 📱 Designed exclusively for **HyperOS 3 (Android 15)**
- ♻️ Systemless installation with easy removal
- 🚀 Premium visual enhancement without real-time blur
- 💯 Stable, clean, and performance-friendly

---

## Requirements

- HyperOS 3
- Android 15
- Root access
- **Magisk** or **KernelSU**

> **Compatibility:** This module is intended **only for HyperOS 3 running Android 15**. It is **not compatible** with HyperOS 1, HyperOS 2, or other Android versions.

---

## Installation

### Magisk

1. Download the latest release.
2. Open the **Magisk** app.
3. Navigate to **Modules**.
4. Tap **Install from Storage**.
5. Select the downloaded ZIP file.
6. Reboot your device.

### KernelSU

1. Download the latest release.
2. Open the **KernelSU Manager** app.
3. Navigate to **Modules**.
4. Install the downloaded ZIP file.
5. Reboot your device.
